# Ferienplaner – Sommerferien 2026

Digitale Version des handgezeichneten Ferien-„Stundenplans" (Mo 22. Juni – So 5. Juli 2026)
zum flexiblen Planen per **Drag & Drop**.

## Öffnen

Einfach `index.html` im Browser öffnen – am Handy, Tablet oder Laptop.
Es braucht **keine Installation und keinen Server**, und es funktioniert auch offline.

## Was geht

- **Termine verschieben:** eine Termin-Karte mit dem Finger/der Maus auf einen anderen Tag ziehen.
- **Ideen einplanen:** aus dem **Ideen-Pool** (eure Lieblings-Ausflüge) eine Idee auf einen Tag ziehen
  – dabei wird ein Termin erstellt, die Idee bleibt für später im Pool.
- **Termin ausplanen:** eine Termin-Karte zurück in den Ideen-Pool ziehen.
- **Bearbeiten/Löschen:** eine Karte antippen – Titel, Tag, Zeit/Notiz, Bemerkung und Farbe ändern.
- **Neu anlegen:** „+ Termin" oder „+ Idee".
- **Farben** entsprechen euren Markern: Rot, Grün, Orange, Blau, Lila.

## Speichern & Teilen

- Alles wird **automatisch im Browser** gespeichert (localStorage), pro Gerät.
- Zum Übertragen auf ein anderes Gerät: **„Sichern"** lädt eine Datei herunter,
  die ihr verschickt und auf dem anderen Gerät über **„Laden"** wieder einlest.
- **„Zurücksetzen"** stellt die Original-Vorlage vom Foto wieder her.

## Vorbefüllung

Die Termine und Ideen wurden vom Foto übertragen (teils handschriftlich, also als beste Lesart).
Alles lässt sich direkt in der App korrigieren – einfach antippen.

## Optional online stellen

Über **GitHub Pages** kann die Seite als Link bereitgestellt werden
(Repo-Einstellungen → Pages → Branch wählen → `index.html` wird zur Startseite).
